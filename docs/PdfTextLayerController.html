PdfTextLayerController

PdfTextLayerController

DOM text overlay for pages with embedded PDF text or external OCR text.

This is deliberately active only on settled spreads. During animation the canvas is the source of truth, and trying to morph real DOM text with the page turn would make selection unreliable.
